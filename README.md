# Teen Mental Health Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes the impact of social media usage on teen mental health, focusing on anxiety, stress, sleep patterns, and addiction levels.

## Objectives
- Analyze anxiety and stress trends
- Study social media usage behavior
- Identify high-risk groups
- Understand sleep and screen-time patterns

## KPIs
- Avg Anxiety %
- Avg Stress %
- Avg Sleep Hours
- Avg Social Media Hours

## Dashboard Features
- Interactive slicers
- Multi-page navigation
- Platform-wise analysis
- Gender and age analysis
- Insights & recommendations

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset

## Key Insights
- High social media usage correlates with increased anxiety levels.
- Students sleeping less than 6 hours show higher stress scores.
- TikTok users reported higher addiction levels compared to other platforms.
